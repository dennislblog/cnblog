Argparse简要用法总结

argparse 是python自带的命令行参数解析包，可以用来方便地读取命令行参数，比如这样

1    parser  = argparse.ArgumentParser()
2    parser.add_argument("--a", type=int, default=16, help="size of the batches")
3    args = parser.parse_known_args()             //args[0] = Namespace(a=16), 也可以直接call args[0].a
参数 说明 例子
default 默认参数 parser.add_argument("--a", default=16)
required 不提供参数会报错 parser.add_argument('-a', required=True)
type 参数类型(默认是str) parser.add_argument("--a", type=int)
choices 不提供选项里的会报错 parser.add_argument("--a", choices=['alexnet', 'vgg'])
help 说明文档 parser.add_argument("--a", help="size of the batches"
dest 设置参数变量名(默认是--或-后面的字符串) parser.add_argument('--a', dest='var_a')
- | - - -后面只能跟一个字符 run -abc == run -a -b -c
nargs 可以提供几个参数 (?=0 or 1, * = 0 or infty, + >= 1) parser.add_argument('-a', nargs='+') 至少跟1个参数, 比如 python tmp.py -a A B C
